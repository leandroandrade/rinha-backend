# rinha-backend

PoC para a rinha de backend.

## Get started

Details to configure local environment:

Installing dependencies:

```shell
npm install
```

Configuring .env file

```shell
cp .env.example .env
```

Starting application like development:

```shell
npm run dev
```

Command to run tests:

```shell
npm t
```

Base url to API:
```
http://localhost:3000/
```
